# 响帮帮日常·测试版

一个可直接导入微信开发者工具的独立测试小程序，用于演示团队日常待办、工作记录和今日概览。

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件所在的 `miniprogram-daily-test`。
4. AppID 可继续使用测试号，项目已配置为 `touristappid`。
5. 点击编译即可预览。

## 测试功能

- 今日工作概览与完成进度。
- 待办按全部、待处理、已完成筛选。
- 新建任务并设置分类、优先级和计划日期。
- 点击任务切换完成状态。
- 添加当天工作记录。
- 本地保存数据，关闭开发者工具后仍可保留。
- 在“我的”页面一键恢复演示数据。

## 说明

- 当前版本不连接网络和业务后端。
- 所有数据仅保存在微信小程序本地缓存中。
- 页面中只包含演示数据，请勿录入真实敏感信息。
